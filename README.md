# Mais où Est(-)elle ?

## Contexte

Projet personnel, dans le but notamment de découvrir Next.JS.

Pour voir le rendu visuel : [mais-ou-estelle.vercel.app/](https://mais-ou-estelle.vercel.app/)

## Description

Passionnée de voyages depuis toujours, je souhaitais créer un blog à mon image, qui récapitule mes itinéraires et mes conseils sur les différentes destinations que j'ai pu découvrir.

## Stack technique

- Next.JS
- Typesript
- Tailwind CSS
- Base de données : utilisation d'une BDD serverless avec API REDIS sur  [upstash.com](https://upstash.com/)
- Déploiement : [vercel.com](https://vercel.com/)
